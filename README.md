## [浙大词典](/浙大词典)

学号简介: 十位数的学号  
后四位代表录取的序号（排名不分先后）  
中间三位固定是010  
再往前两位代表入学年份  
比如24代表2024年入学的学生  
第一位是3，代表浙大是top3（樂）
